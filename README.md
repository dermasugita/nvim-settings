# nvim-settings

## Plugin Manager
- lazy vim

## Options Location
- ~/.config/nvim

## Key Binds
### Insert Mode
- jj to <Esc>

### Terminal Mode
- <Option-k> to <Ctrl-\><Ctrl-n><Ctrl-w>k # to the upper window
- <Option-h> to <Ctrl-\><Ctrl-n><Ctrl-w>h # to the left window
- <Option-n> to <Ctrl-\><Ctrl-n> # to terminal normal mode
